# My CV using Typst

> My CV using [typst](https://github.com/typst/typst)

Adapted from [Modern Academic CV](https://github.com/bpkleer/typst-modern-acad-cv)

## Install

```zsh
brew install typst
```

## Compile

```zsh
# Compile 
typst compile Kittipos-CV.typ
# Watch
typst watch Kittipos-CV.typ
```
